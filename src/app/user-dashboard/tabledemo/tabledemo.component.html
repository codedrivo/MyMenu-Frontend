User Assistance

Before proceeding with the upload of your report template, it is crucial to ensure that the following details have been correctly entered:

(Note: Ensure that in the uploaded menu template, the Menu column contains one of the following values: Breakfast, Lunch, or Dinner)

  • Menu
  • Menu Item Name

These details are essential for the generation of your report and must be filled out with precision within the template.

Mandatory Information:

  • Menu: This field is non-negotiable for validation to be successful.
  • Menu Name: This field is non-negotiable for validation to be successful.
  • Menu Item Description: Can be mentioned if required
  • Menu Item Category: Can be mentioned if required
  • Menu Item Current Price: Can be mentioned if required
  • Menu Item Promotional Price: Can be mentioned if required
  • Menu Item Promotional Description: Can be mentioned if required
  • Calories: Can be mentioned if required
  • Top Selling Item (Yes/No): Can be mentioned if required

In the absence of a specified Distance From Zip Code, a default radius of 25 miles will be applied, this should be entered while getting registered
